# Login & Signup Form

🔗 **Live Demo:** [https://PranjalChirde.github.io/Login-signup-Form/](https://PranjalChirde.github.io/Login-signup-Form/)

A simple React app with Login and Signup forms built while learning React form handling.

## Features

- Signup form with Username, Email, and Password
- Login form with Email and Password
- Switch between Login and Signup pages
- Controlled components using `useState`
- Form reset after submission

## Tech Stack

- React
- Vite
- CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm run dev
```

App runs at: `http://localhost:5173`

## What I Learned

- Controlled form components in React
- Handling form input with `useState`
- Passing props between components
- Navigating between components using state
